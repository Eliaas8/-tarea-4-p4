#  Biblioteca Personal con MongoDB

##  Descripción
Aplicación de línea de comandos en Python que gestiona una biblioteca personal utilizando MongoDB como base de datos no relacional.

##  Tecnologías
- Python
- MongoDB
- pymongo

##  Instalación de MongoDB

### Opción 1: MongoDB local
Descargar desde:
https://www.mongodb.com/try/download/community

Ejecutar servicio:
```bash
mongod
